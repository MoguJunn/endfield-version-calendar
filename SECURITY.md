# 安全策略

## 支持范围

安全修复只针对 `main` 分支的最新版本。历史部署和个人分支不在维护范围内。

## 报告安全问题

请使用 GitHub 的私密漏洞报告功能：

<https://github.com/MoguJunn/endfield-version-calendar/security/advisories/new>

请尽量提供：

- 问题类型和潜在影响
- 可复现步骤或最小示例
- 受影响的页面、文件或提交
- 建议修复方式（如有）

请不要在公开 Issue 中披露尚未修复的漏洞、令牌、私钥或其他敏感信息。维护者会在确认报告后协调修复与公开时间。

## 项目安全边界

本仓库是静态前端，不应包含数据库高权限密钥、服务器凭据或生产环境变量。浏览器只访问主站公开 API；任何需要高权限凭据的功能都不应在本仓库中实现。
