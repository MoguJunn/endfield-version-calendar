03 / PUBLIC API

公开活动 API

免密、只读并允许跨域调用，返回日历页面使用的最终合并活动数据。

完整接口说明 → 下载 OpenAPI 3.1 ↓
GET https://ef-cal.mogujun.icu/api/v1/events

默认返回全部已整理版本、活动、角色寻访与武库申领。

调用示例 fetch("https://ef-cal.mogujun.icu/api/v1/events?version=5&status=live")

按 Version 5 和“进行中”状态筛选活动。

version 版本号或版本键 category 一个或多个活动分类 status upcoming / live / ended from / to 带时区的时间范围

默认按客户端 IP 每 60 秒最多处理 60 次函数请求；超限返回 HTTP 429。成功响应允许共享缓存 60 秒，错误响应不会缓存。
